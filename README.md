---
## Running the Project on Your Machine

#### Create a Folder & Clone the Repository.
```
mkdir ~/Dev/svelte-project
cd ~/Dev/svelte-project
```

#### Install Dependencies
```
npm install
```

**For yarn  type: `yarn install`**

#### Run Dev Server
```
npm run dev
```

#### Build for Production
```
npm run build
```
